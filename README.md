# Nuxt Develop - Starter Pack

Look at the [nuxt 3 documentation](https://v3.nuxtjs.org) to learn more.

## Setup

```bash
# Install
npm install

# Start the development server on http://localhost:3000
npm run dev

# Generate
npm run generate
```
